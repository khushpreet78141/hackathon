# 🧠 ShadowInfluence — AI-Based Persuasion & Manipulation Detector

> Detect hidden psychological influence, propaganda, and manipulation in text and images using AI.

---

## 🔍 Overview

**ShadowInfluence** is an AI-powered analysis platform that identifies **manipulative patterns**, **emotional persuasion**, and **propaganda signals** in digital content.

It combines **Natural Language Processing (NLP)** and **pattern detection models** to provide **real-time influence scoring, analytics, and visualization**.

---

## ⚡ Key Features

### 🧠 AI Influence Detection

* Detects **emotional manipulation**
* Identifies **propaganda patterns**
* Highlights **persuasive language cues**

### 📊 Analytics Dashboard

* **Influence Score (0–100%)**
* Interactive **charts & visual insights**
* Trend tracking of manipulation over time

### 🔥 Influence Heatmap

* Highlights **high-risk sentences**
* Shows **where manipulation occurs in text**

### 🧩 Psychological Pattern Detection

* Fear-based persuasion
* Authority bias
* Urgency & scarcity triggers

### 📈 Influence Trend Tracking

* Tracks how manipulation evolves across content
* Useful for media analysis & research

### 🗂️ History & Logs

* Stores recent analyzed texts in database
* Displays insights & scan history

### 🌐 Browser Extension (🔥 Unique Feature)

* Detects manipulation **in real-time on websites**
* Flags:

  * Fake news
  * Biased content
  * Persuasion tactics

## 🧠 How It Works

1. User submits text or uploads content
2. Backend processes using AI models
3. System detects:

   * Emotional triggers
   * Manipulative phrases
   * Propaganda signals
4. Generates:

   * Influence Score
   * Heatmap
   * Analytics charts
5. Stores results for future tracking
